### 📋 概要

GradioベースのAWS対応猫登録システム（日本語UI）です。シンプルで使いやすい猫の情報管理を実現します。

</div>


https://github.com/user-attachments/assets/c81d67c6-e290-498e-890b-844124e8db1d

## 主な機能
- 猫の基本情報登録（名前、年齢、品種、写真）
- 登録済み猫の一覧表示と管理
- AWS S3を使用した画像保存
- AWS RDS PostgreSQLでのデータ管理
